# 06_CONFIGURATION_MANAGEMENT - Configuration Management Records

**⚠️ DOCUMENTATION LEVEL:** [Level 1] AISET Tool Development Only

---

## Purpose

This folder contains **AISET tool configuration management data** per DO-178C DAL D.

**⚠️ CRITICAL:**
- This is for **AISET tool CM** (managing AISET development baselines and changes)
- **NOT for user system CM** (what AISET helps users manage)

---

## Contents

### Subdirectories
- `Baseline_Records/` - AISET baseline records
- `Change_Requests/` - AISET change requests
- `Problem_Reports/` - AISET problem reports
- `Version_Control_Logs/` - AISET version control history
- `Tool_Baselines/` - AI tool version baselines used for AISET development
