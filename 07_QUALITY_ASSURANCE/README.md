## Purpose

This folder contains **AISET tool quality assurance data** per DO-178C DAL D.

**⚠️ CRITICAL:**
- This is for **AISET tool QA** (ensuring quality of AISET development)
- **NOT for user system QA** (what AISET helps users implement)

## Contents

- `Process_Compliance_Records/` - AISET compliance with DO-178C process

- `Non_Conformance_Reports/` - AISET NCRs and corrective actions
